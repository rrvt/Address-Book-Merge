Build Addr Module

Intruduction

The Thunderbird Address Book can export one or more of its address books. This application will create a module with a record with a field for every field in the CSV file. The names of the fields in the record come from the first line of the address book csv file. The module, addr.h/cpp, is added to AddrBookMerge project to implement the the merge application. It implements a few simple operations describe under operations.
