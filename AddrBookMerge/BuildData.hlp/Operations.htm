Build Addr Module

Open File

Opens a file that has been exported from Thunderbird's Address Book. The header line of the CSV file is read and the names in the header line are used for the fields in the Addr Module Record.

Pick eMail Field Name

A dialog is presented with the names of all the fields in the record. The user must choose the field that holds the eMail address in the record. At the time of writing it is Primary_Email. This fields is used to sort the address lines during the processing of the records.

Prepare Addr Module Header File

Use the information from the header line and the picked email field to prepare Addr.h, the header file of the Addr Record Module. Setup the Save File command to save the header file to "Addr.h".

Prepare Addr Module Body File

Same as the header file preparation but for the body file of the module, Addr.cpp.

Save File

Displays a Save File Dialog and saves the current file displayed in the window.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

Print

Will do all the normal things to print a file.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
