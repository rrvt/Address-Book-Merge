Address Book Merge Operations

Open an Address Book

Opens an Address Book CSV File and displays it in the window. Scroll bar works to see the whole file.

Add an Address Book

Add an Address Book CSV File to the app and display all of them.

Process Address Books

After loading one or more address books into the application, process it in a variety of ways to remove unwanted eMail addresses, fix misfiled eMail addresses, merge duplicate eMail records and remove duplicates.

Save File

Displays a Save File Dialog and saves the address book CSV file to the file named.

Print

Will do all the normal things to print a file.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Exit the application with no reservations.
