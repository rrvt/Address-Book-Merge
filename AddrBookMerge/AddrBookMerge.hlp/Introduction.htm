Address Book Merge

Introduction

Thunderbird collects eMail addresses with each eMail. Thunderbird has an Address Book Export function. Export all the books, do some processing then output a new Address Book. Delete the current address books in Thunderbird and import the new one. The goal is to remove unwanted eMail addresses, missing eMail addresses, fix eMails that are misplaced in the fields and finally merge and remove duplicates. It implements a few simple operations describe under operations.
